# Paper build

This folder contains a LaTeX manuscript that embeds the plots and tables generated by the experiments.

## 1) Ensure results exist

From repo root:

```cmd
.venv\Scripts\python scripts\run_experiments.py --experiment bitflip --shots 2000
.venv\Scripts\python scripts\run_experiments.py --experiment phaseflip --shots 2000
.venv\Scripts\python scripts\run_experiments.py --experiment depolarizing --shots 2000
```

## 2) Generate LaTeX tables from CSV

```cmd
.venv\Scripts\python paper\make_tables.py
```

This writes `paper/tables/*.tex`.

## 3) Compile PDF

If you have TeX Live / MiKTeX installed:

```cmd
pdflatex -interaction=nonstopmode -output-directory paper paper\main.tex
pdflatex -interaction=nonstopmode -output-directory paper paper\main.tex
```

Alternative (if you have latexmk):

```cmd
latexmk -pdf -output-directory=paper paper\main.tex
```
